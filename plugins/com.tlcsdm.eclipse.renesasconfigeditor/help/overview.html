Introduction

This plugin registers files—commonly used in Renesas projects—to be opened by Eclipse’s built-in XML editor. It helps developers quickly view and edit project configuration files in a structured and readable way.

This plugin supports the following project file format:

  • mtpj
  • scfg
  • ewp
  • inf
